# SpookyFC Flight Controller
Spooky FC is an open-source, INAV-compatible flight controller built as a practical learning platform for designing and experimenting with custom drone electronics. Powered by the STM32F405 MCU, it integrates modern motion sensors, multiple UART interfaces, and a robust, flexible power architecture. Spooky FC is designed to be approachable for beginners while still offering enough depth and expandability for hardware hackers and enthusiasts who want to explore flight controller design from the ground up

## Features

- STM32F405 MCU for reliable, high-performance flight control

- INAV-compatible firmware support

- Modern IMU sensors for stable and accurate flight

- Multiple UARTs for GPS, telemetry, receiver, and peripherals


- Flexible power system supporting common drone power setups

- Open-source hardware & firmware friendly

- Beginner-friendly design with room for experimentation and mods
